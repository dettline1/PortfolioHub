┌──────────────────────────────────────────────────────────────────────┐
│  ⚙️ ОПЦИОНАЛЬНО: GitHub Token                                        │
└──────────────────────────────────────────────────────────────────────┘

  Зачем нужен?
    • Без токена: 60 запросов/час
    • С токеном: 5000 запросов/час

  Как получить?
    1. https://github.com/settings/tokens
    2. Generate new token (classic)
    3. Scopes: public_repo
    4. Скопируйте токен

  Куда добавить?
    Откройте .env и вставьте:
    GITHUB_TOKEN=ghp_ваш_токен_здесь

  Подробнее: GITHUB_INTEGRATION.md
